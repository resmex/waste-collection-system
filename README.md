# waste-collection-system
A web-based waste collection system using PHP, MySQL and GPS Tracking.

step-by-step guide for installation and usage.

1. Installation (Localhost / XAMPP)
    Step 1 — Download the project
   git clone https://github.com/resmex/waste-collection-system.git

   Or download ZIP and extract it into:
   C:\xampp\htdocs\waste-collection-system

   Step 2 — Create database
   Open phpMyAdmin --> Click New --> Create database(name: wms)

   Step 3 — Import SQL File
   Go to Import --> Select the SQL file from: database/wms.sql -->Click Go

   Step 4 — Start the System
   Open XAMPP --> Start Apache --> Start MySQL --> open your browser:    http://localhost/waste-collection-system/
